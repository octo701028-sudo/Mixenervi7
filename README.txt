Enervi7 黏著性升級 Patch
================================

可直接丟到 GitHub Pages 的極簡前端版本（無外部函式庫）。
包含：
- 7 個滑桿快速測驗
- 純 SVG 雷達圖
- 當日呢喃神諭（避免重複）
- 依最低分階段給出行動建議
- 自動展開最低分階段卡（手風琴）
- 連續打卡（7 / 21 / 77 日徽章）
- 近 7 日歷史記錄（localStorage）

使用方式
--------
1) 把以下檔案放到你的 repo 根目錄：
   - index.html
   - styles.css
   - main.js
   - manifest.webmanifest（可選）

2) 如果你的現有專案有自己的樣式或檔名，
   你可以把 main.js 內容合併進原本的 JS，然後將
   HTML 中的結構貼入合適位置。

3) 此版本不依賴任何 CDN，直接可用於 GitHub Pages。
